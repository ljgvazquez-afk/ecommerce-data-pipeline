# Ecommerce Data Pipeline

## Overview

Data Engineering project built using Python, Google Cloud Platform, BigQuery and Cloud Storage.

The project extracts data from multiple REST APIs, applies ETL transformations, validates data quality and stores datasets for analytics and business reporting.

## Architecture

Products API
→ Python ETL
→ Cloud Storage
→ BigQuery
→ SQL Analytics

Users API
→ Python ETL
→ Cloud Storage
→ BigQuery
→ SQL Analytics

Orders API
→ Python ETL
→ Cloud Storage
→ BigQuery
→ SQL Analytics
